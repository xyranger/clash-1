# Features

- Support Vmess/Shadowsocks/Socks5
- Support Shadowsocks-V2ray-plugin
- Support Shadowsocks-Obfs-plugin
- HTTP/HTTPS and SOCKS protocol
- Support Custom Rule
- Support fake-ip
